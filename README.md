# Next.js Blogging Platform - Latest Auto Tech

This repository is a starter scaffold for a modern blogging platform using Next.js (App Router), Tailwind CSS, and MongoDB with Mongoose.

Features included:
- App Router structure with sample pages and components
- MongoDB models and connection helper
- API routes for auth, blogs, categories, comments, subscribers
- Admin user seed script

Setup:
- Install dependencies: `npm install`
- Create `.env` with `MONGODB_URI` and `JWT_SECRET`
- Run dev: `npm run dev`
- Seed sample data: `npm run seed`
